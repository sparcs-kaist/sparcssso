# SPARCS SSO

SPARCS SSO provides integrated login in SPARCS services, such as ARA or OTL.

SPARCS SSO uses following program / framework / libraries:
* Python 3.9.5
* Django 3.2.4
* Django REST Framework 3.12.4
* MySQL (or MariaDB)
* Bootstrap 3.3.x
* jQuery 1.11.x
